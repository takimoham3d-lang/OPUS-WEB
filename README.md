# Opus FSSM
 
Site web du club musique et arts de la scène **Opus FSSM**, réalisé dans le cadre d'un projet web.
 
## Lien vers le site
 
https://takimoham3d-lang.github.io/OPUS-WEB/index.html
 
## Programmes
 
HTML, CSS, JavaScript
 
## Structure
 
```
opus/
├── index.html        — Page d'accueil
├── about.html        — À propos du club
├── activities.html   — Activités
├── events.html       — Événements
├── join.html         — Rejoindre le club
└── assets/
    ├── css/style.css
    ├── js/main.js
    └── images/
```
 
## Lancer le projet
 
Ouvrir `index.html` dans un navigateur.
 
## Fonctionnalités
 
- Présentation du club et de ses activités
- Page événements
- Formulaire d'adhésion
- Design responsive
